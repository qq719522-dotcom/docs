# iqqz API 文档

iqqz API 兼容 OpenAI 风格接口，适合接入聊天、图片、视频、音频、OCR 和 3D 等能力。

## 推荐阅读

- [前言-模型兼容性（必读）](/vapi/5069242m0)
- [OpenAI官方SDK使用教程](/vapi/5071665m0)
- [高并发批量请求示例](/vapi/5544907m0)

## 常用入口

- 接口基址：`https://api.iqqz.com/v1/`
- 身份凭证：在 iqqz 控制台获取 API Key
- 兼容性：OpenAI 官方库可直接使用

## 覆盖内容

- 聊天模型与模型列表
- 图片、视频、音频、音乐生成
- OCR、PDF、3D 与内容处理
