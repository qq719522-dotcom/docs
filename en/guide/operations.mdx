## Scenario 1: Find the right feature

1. Open the matching module from the top navigation or sidebar.
2. Confirm the page title matches what you want to do.
3. Continue with the form or action on the page.

## Scenario 2: Fill in information

Use this order when filling a form:

1. Complete the required fields first
2. Add optional details second
3. Review everything before submitting

## Scenario 3: Confirm the result

After submission, you will usually see one of these:

- A success message
- A page redirect
- A new record in a list

If you do not see the result, refresh once and check again.

## Scenario 4: Edit existing content

If you need to change something after submission, look for:

- An edit button
- A resubmit action
- A cancel or rollback option

## Scenario 5: Mobile usage

On mobile, keep in mind:

- Menus may be hidden behind a hamburger button
- Inputs are tighter, so double-check before submitting
- Complex actions are often easier on desktop
