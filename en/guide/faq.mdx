## Why can't I see a feature?

Common reasons:

- Your account may not have permission
- The feature may only appear in specific contexts
- The page may need a refresh

## What should I do if login fails?

Check the following first:

- Username or password
- Expired verification code
- Network connectivity

If it still doesn't work, try password recovery or contact support.

## What if nothing happens after I submit?

Try this order:

1. Look for an error message
2. Refresh to confirm whether it submitted
3. Check for missing required fields
4. Try another browser

## Is mobile the same as desktop?

The workflow is usually the same, but the layout may differ.

If the action is complex, desktop is usually easier.
